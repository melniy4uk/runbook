# Зачем этот проект

Runbook — CLI/TUI-утилита, которая превращает часто используемые команды
терминала в понятные сценарии запуска. Вместо того чтобы каждый раз вспоминать
и собирать длинные команды вручную, ты выбираешь готовый сценарий и запускаешь
его одинаково и предсказуемо.

## Чем помогает

- Ускоряет работу: запуск команд по имени сценария вместо ручного ввода.
- Снижает количество ошибок: меньше опечаток и перепутанных аргументов.
- Стандартизирует процессы: единый способ запускать типовые задачи в команде и
  в проекте.
- Делает запуск прозрачным: заранее понятно, что именно будет выполнено.
- Повышает повторяемость: одинаковые входные параметры → одинаковый результат.
- Упрощает вариативность: удобно переключать заранее подготовленные варианты
  параметров.

## Сценарий использования

1. Один раз описываешь типовые команды как сценарии: сборка, тесты, линтинг,
   деплой, запуск сервисов, утилиты, миграции и т.д.
2. В CLI или TUI открываешь список доступных сценариев и выбираешь нужный.
3. При необходимости меняешь несколько параметров для конкретного запуска.
4. Запускаешь сценарий и получаешь ожидаемый результат.
5. Повторяешь запуск в любой момент — без “вспомнить команду” и без ручной
   пересборки аргументов.
