Сценарий использования

1. Я добавляю файлы с командами в каталог Runbook (в одном файле может быть несколько команд).


2. Runbook читает все файлы и собирает общий список команд.


3. Я запускаю Runbook (CLI или TUI) и выбираю нужную команду.


4. Runbook показывает переменные этой команды — я выбираю/ввожу значения.


5. Runbook собирает и показывает конечную команду.


6. Я подтверждаю запуск — команда выполняется.
